# Topic: IP Addressing Basics

## Objective
Understand what IP addresses are and how they identify devices on a network.

## Key Concepts
- IP address
- Public vs Private IP
- IPv4

## What I Learned
- An IP address uniquely identifies a device on a network
- Private IPs are used inside local networks
- Public IPs are used to communicate over the internet

## Why This Matters in Cybersecurity
- Attackers and defenders both track IP addresses
- Logs often contain IP addresses for investigation
- Understanding IPs helps in identifying suspicious traffic
